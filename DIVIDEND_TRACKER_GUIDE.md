**檔案位置**: `/home/administrator/.openclaw/workspace/stock_web/dividend_tracker.html`

## 📊 自動計算功能

### **1. 年股息計算**

```
年股息 = 持有股數 × 每股年配息

範例：
持有 00919 4,000 股
每股年配息 $4.25
→ 年股息 = 4,000 × $4.25 = $17,000
```

---

### **2. 實際殖利率計算**

```
實際殖利率 = (年股息 ÷ 投入金額) × 100%

範例：
年股息 $17,000
投入金額 $200,000
→ 實際殖利率 = ($17,000 ÷ $200,000) × 100% = 8.5%
```

---

### **3. 健保狀態判斷**

```
if 年股息 ≥ 20,000:
    顯示「⚠️ 超過 20,000（需扣 1.91%）」
elif 年股息 ≥ 18,000:
    顯示「⚠️ 接近 20,000」
else:
    顯示「✅ 安全範圍」
```

---

### **4. 平均殖利率計算**

```
平均殖利率 = (總年股息 ÷ 總投入金額) × 100%

範例：
總年股息 $51,610
總投入金額 $887,000
→ 平均殖利率 = ($51,610 ÷ $887,000) × 100% = 5.82%
```

## ⚠️ 重要提醒

### **1. 資料為參考值**

- 殖利率、股價、配息都是**估算值**
- 實際配息會波動
- 建議定期查詢最新公開說明書

### **2. 非投資建議**

- 本系統為**學習工具**
- 不構成投資建議
- 投資前請評估自身風險承受能力

### **3. 健保補充保費**

- 門檻為 20,000 元（2024 年）
- 費率為 1.91%
- 規則可能調整，請查詢最新健保署公告
