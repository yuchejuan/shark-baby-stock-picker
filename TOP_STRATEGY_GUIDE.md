# 🦈 TOP 選股策略完整指南

## 🎯 策略核心理念

**目標**: 從台股市場中，每天篩選出技術面最強、風險最低的 TOP 10 股票

**方法**: 使用 **6 大技術指標** 綜合評分，自動化選股

**特色**: 
- ✅ 100% 量化選股（無人為主觀判斷）
- ✅ 多指標共振（降低假訊號）
- ✅ 每天自動更新（06:00 執行）
- ✅ 歷史績效可追蹤

## 📊 六大技術指標解析

### 1. RSI（相對強弱指標）- 15 分

**作用**: 判斷股票是否「超買」或「超賣」

**計算**: 14 天內上漲天數 vs 下跌天數的比率

**評分邏輯**:
```
RSI < 30  → 超賣（嚴重低估）→ +15 分 🟢 強烈買點
RSI < 40  → 偏低（可能反彈）→ +10 分 🟡 留意
RSI 40-60 → 中性（正常波動）→ +5 分  ⚪ 觀察
RSI 60-70 → 偏高（小心）     → 0 分   🟠 謹慎
RSI > 70  → 超買（可能回檔）→ -10 分 🔴 風險
```

**實際範例**（今日 2353 宏碁）:
```
RSI = 58.12
判斷: 中性偏高（接近 60）
得分: +5 分
解讀: 不算超買，但也不是超賣好時機
```

---

### 2. MACD（平滑異同移動平均線）- 15 分

**作用**: 判斷「趨勢方向」和「轉折點」

**計算**: 
- DIF（快線）= EMA12 - EMA26
- DEA（慢線）= DIF 的 9 日移動平均
- 柱狀圖 = DIF - DEA

**評分邏輯**:
```
DIF > DEA 且向上 → 多頭趨勢        → +10 分 🟢
DIF 向上穿越 DEA → 黃金交叉（買訊）→ +15 分 🟢🟢
DIF < DEA 且向下 → 空頭趨勢        → -10 分 🔴
DIF 向下穿越 DEA → 死亡交叉（賣訊）→ -15 分 🔴🔴
```

**實際範例**（今日 2353 宏碁）:
```
MACD = "多頭"
解讀: DIF > DEA，趨勢向上
得分: +10 分
推薦理由: "MACD黃金交叉" → 最近剛轉多頭
```

**圖解**:
```
價格走勢圖
  ↑
  |     ╱
  |   ╱   ╱
  |  ╱  ╱     ← DIF（快線）
  | ╱ ╱
  |╱╱        ← DEA（慢線）
  ╱
  |
  └─────────→ 時間

當 DIF 向上穿越 DEA = 黃金交叉 = 買點！
```

---

### 3. KD 指標（隨機指標）- 12 分

**作用**: 判斷「短期超買超賣」

**計算**: 
- K 值 = 當前價格在近期區間的位置
- D 值 = K 值的移動平均

**評分邏輯**:
```
K < 20, D < 20  → 超賣（短線買點）  → +12 分 🟢
K 向上穿 D      → 黃金交叉          → +10 分 🟢
K > 80, D > 80  → 超買（短線賣點）  → -12 分 🔴
K 向下穿 D      → 死亡交叉          → -10 分 🔴
K 與 D 在 50 附近 → 中性            → +5 分  ⚪
```

**實際範例**（今日 2353 宏碁）:
```
KD = "超賣"
解讀: K 或 D 值低於 20
得分: +12 分
推薦理由: "KD超賣" → 短線有反彈機會
```

**KD + MACD 共振**:
```
2353 宏碁:
- MACD 黃金交叉（中長期轉多）
- KD 超賣（短線買點）
→ 多指標共振！評分高達 83 分！
```

---

### 4. 布林通道（Bollinger Bands）- 12 分

**作用**: 判斷價格是否「偏離正常區間」

**計算**:
- 中軌 = 20 日移動平均
- 上軌 = 中軌 + 2 倍標準差
- 下軌 = 中軌 - 2 倍標準差

**評分邏輯**:
```
價格 < 下軌    → 超跌（可能反彈）  → +12 分 🟢
價格接近下軌   → 偏低              → +8 分  🟡
價格在中軌附近 → 正常              → +5 分  ⚪
價格接近上軌   → 偏高              → 0 分   🟠
價格 > 上軌    → 超漲（可能回檔）  → -12 分 🔴
```

**圖解**:
```
價格走勢
  ↑
  |─────────  上軌（超漲警戒線）
  |    ╱╲╱
  | ─╱─────  中軌（20日均線）
  |╱    ╲
  |───────── 下軌（超跌買點線）
  |
  └──────→ 時間

當價格碰到下軌 = 超跌 = 買點！
當價格碰到上軌 = 超漲 = 賣點！
```

---

### 5. 移動平均線（MA）- 15 分

**作用**: 判斷「長期趨勢方向」

**使用**:
- MA5（週線）= 5 日均價
- MA20（月線）= 20 日均價
- MA60（季線）= 60 日均價

**評分邏輯**:
```
MA5 > MA20 > MA60 → 多頭排列（強勢）→ +15 分 🟢🟢
價格站上 MA20     → 突破月線         → +8 分  🟢
MA60 > MA20 > MA5 → 空頭排列（弱勢）→ -15 分 🔴🔴
價格跌破 MA60     → 破季線（危險）  → -10 分 🔴
```

**實際範例**（今日 2353 宏碁）:
```
MA5  = 27.84
MA20 = 27.22
MA60 = 26.65
現價 = 27.15

判斷: MA5 > MA20 > MA60 → 多頭排列！
得分: +15 分
解讀: 短、中、長期均線都向上，趨勢明確向上
```

**圖解**:
```
價格走勢
  ↑
  |        ╱ MA5（最靈敏）
  |      ╱
  |    ╱   MA20（月線）
  |  ╱
  |╱      MA60（季線）
  |
  └──────→ 時間

多頭排列 = 三線向上 = 趨勢強勁！
```

### 6. OBV（能量潮指標）- 10 分

**作用**: 判斷「成交量趨勢」（資金動向）

**計算**: 
- 如果今天收盤 > 昨天 → OBV += 今日成交量
- 如果今天收盤 < 昨天 → OBV -= 今日成交量

**評分邏輯**:
```
OBV ↑ + 價格 ↑ → 量價齊揚（資金進場）→ +10 分 🟢
OBV ↑ + 價格 ↓ → 背離（可能反轉）    → +5 分  🟡
OBV ↓ + 價格 ↑ → 背離（虛漲）        → -5 分  🟠
OBV ↓ + 價格 ↓ → 賣壓大（續跌）      → -10 分 🔴
```

**實際意義**:
```
OBV 上升 = 資金持續買進 = 有人看好 = 好訊號
OBV 下降 = 資金持續賣出 = 有人看壞 = 壞訊號
```

## 🎯 綜合評分系統

### 總分計算

```
總分 (100分) = 基礎分 (30) + 技術指標分 (70)
```

#### 基礎分（30 分）
- **價格穩定性**（10 分）：波動率不能太大
- **成交量活躍度**（10 分）：要有足夠流動性
- **流動性**（10 分）：市值夠大，易買易賣

#### 技術指標分（70 分）
- RSI（15 分）
- MACD（15 分）
- KD（12 分）
- 布林通道（12 分）
- 移動平均線（15 分）
- OBV（10 分）

---

```
   → 短線買點（+12 分）
```

## 💡 推薦理由解讀

### 常見推薦理由

#### 1. "MACD黃金交叉 + KD超賣"（最強組合）
```
意義：
- MACD 黃金交叉 = 中長期趨勢轉多
- KD 超賣 = 短線已跌深，隨時反彈

實戰價值：★★★★★
- 這是「多週期共振」
- 大趨勢向上 + 短線超賣 = 絕佳買點
- 勝率通常 > 70%

範例：2353 宏碁、2324 仁寶
```

#### 2. "MACD黃金交叉"
```
意義：
- 趨勢從下跌轉為上漲
- 動能開始增強

實戰價值：★★★★
- 適合中期持有（2-4 週）
- 建議配合其他指標確認

範例：2838 聯邦銀、2887 台新金
```

#### 3. "RSI超賣反彈機會"
```
意義：
- RSI < 30，嚴重超賣
- 可能出現「報復性反彈」

實戰價值：★★★
- 適合短線操作（1-2 週）
- 風險：可能「越賣越低」
- 建議搭配止損

範例：2618 長榮航（假設 RSI 20.6）
```

#### 4. "價格突破均線"
```
意義：
- 股價站上 MA20 或 MA60
- 趨勢可能轉強

實戰價值：★★★
- 適合趨勢追蹤
- 注意「假突破」風險

範例：2353 宏碁突破 MA20
```

#### 5. "多頭排列"
```
意義：
- MA5 > MA20 > MA60
- 短、中、長期都向上

實戰價值：★★★★★
- 趨勢最明確
- 適合中長期持有
- 只要不破季線，可續抱

範例：大多數高評分股票
```

## 📈 選股策略建議

### 策略 1：穩健型（推薦新手）

**選股原則**：
- 只買評分 **≥ 75 分**的股票
- 優先選「MACD黃金交叉 + KD超賣」
- 避開評分 < 70 分的股票

**範例組合**（今日）:

## ⚠️ 風險提示

### 系統限制

1. **技術分析不是萬能**
   - 只能判斷「過去」和「現在」
   - 無法預測「突發事件」（例如：公司爆雷、產業政策變化）

2. **假訊號存在**
   - 即使評分 80+，也可能下跌
   - 建議設定停損（-5% 或 -10%）

3. **不考慮基本面**
   - 系統不看財報、營收、EPS
   - 如果公司基本面很差，技術面再好也有風險

4. **市場環境影響**
   - 大盤崩跌時，個股難獨善其身
   - 建議觀察大盤（台股加權指數）

---

### 使用建議

**✅ 建議這樣用**：
```
1. 每天早上看 TOP 推薦
2. 選 2-3 支評分 ≥ 75 的股票
3. 模擬買入（記錄買入理由）
4. 設定停損 -5%
5. 持有 1-4 週
6. 達目標或停損就賣出
7. 月底檢討績效
```

**❌ 不建議這樣用**：
```
1. All-in 單一股票
2. 不設停損
3. 追高殺低（看到漲才買，跌了就賣）
4. 過度交易（每天換股）
5. 忽略大盤走勢
```
