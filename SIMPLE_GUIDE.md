## 📱 三大頁面

### 1. 📊 TOP 選股

**功能：顯示今日推薦股票，可直接買入**

#### 顯示內容
- 排名（前 10 名，🥇🥈🥉）
- 股票代號、名稱
- 目前股價
- 評分（綠色=優良，橘色=普通，紅色=風險）
- 推薦理由（技術分析）
- 買入按鈕

#### 操作流程
```
1. 點擊「📈 買入」按鈕
2. 彈出視窗顯示：
   - 股票資訊
   - 預設股數（1000 股，可修改）
   - 推薦理由（自動填入，唯讀）
   - 個人附註（選填，可加自己的想法）
3. 點「確認買入」
4. 成功提示 → 自動更新持倉
```

#### 範例
```
點擊：宏碁 (2353) @ $27.15 的「買入」

彈窗內容：
股票: 宏碁 (2353) @ $27.15
股數: 1000
買入理由: MACD黃金交叉 + KD超賣（自動填入）
個人附註: [空白，可填「看好未來趨勢」]

→ 確認買入
→ ✅ 成功！總金額 $27,150
```

### 2. 💼 持倉明細

**功能：顯示即時持股與損益**

#### 顯示內容

**頂部總覽**：
- 總成本
- 目前市值
- 總損益（綠色=獲利，紅色=虧損）
- 報酬率 %

**持倉表格**：
| 代號 | 名稱 | 股數 | 買入價 | 現價 | 損益 | 報酬率 | 買入日期 | 持有天數 | 建倉理由 | 操作 |
|------|------|------|--------|------|------|--------|----------|----------|----------|------|

#### 特色
1. **分批次顯示** - 同一股票不同天買入會分開顯示
2. **即時損益** - 自動計算每批次的盈虧
3. **持有天數** - 顯示從買入至今的天數
4. **完整理由** - 顯示「推薦理由 + 個人附註」

#### 操作流程
```
1. 點擊「📉 賣出」按鈕
2. 輸入要賣出的股數
3. 確認
4. 成功提示 → 自動更新持倉
```

#### 範例
```
持倉顯示：
2353 宏碁 1000股 $27.15 → 現價 $28.50
損益: +$1,350  報酬率: +4.97%
買入日期: 2026/3/25  持有: 5天
建倉理由: MACD黃金交叉 + KD超賣 | 附註: 看好未來趨勢

→ 點「賣出」
→ 輸入 1000 股
→ ✅ 成功！獲利 $1,350
```

### 3. 📜 歷次買賣

**功能：完整交易記錄，永久保存**

#### 顯示內容
| 日期 | 類型 | 代號 | 名稱 | 股數 | 價格 | 金額 | 備註 |
|------|------|------|------|------|------|------|------|

- **類型標籤**：
  - 🟢 買入（綠色）
  - 🔴 賣出（紅色）
- **金額顏色**：
  - 買入 = 紅色（花錢）
  - 賣出 = 綠色（賺錢）

#### 操作
```
點擊「💾 匯出 CSV」→ 自動下載 CSV 檔案
```

**CSV 內容**：
```
日期,類型,代號,名稱,股數,價格,金額,備註
2026/3/25,買入,2353,宏碁,1000,27.15,27150,"MACD黃金交叉 + KD超賣 | 附註: 看好未來趨勢"
2026/3/30,賣出,2353,宏碁,1000,28.50,28500,"模擬賣出"
```

可在 Excel 開啟，做進一步分析。

## 🔧 技術說明

### 資料來源
1. **推薦股票**: `daily_report.json`（每天 06:00 更新）
2. **即時股價**: `portfolio.json`（每天 15:30 更新）
3. **交易記錄**: `stock_trades.db`（SQLite，永久保存）

### API 端點
- `GET /api/holdings/batches` - 取得分批次持股
- `POST /api/trade/add` - 新增交易
- `GET /api/trades` - 取得所有交易記錄

### 自動化
- 每天 06:00 → 產生今日推薦
- 每天 15:30 → 更新即時股價
- 即時計算 → 損益、報酬率、持有天數

## 📞 常見問題

### Q: 如果我想看回舊版怎麼辦？

A: 舊版已備份為 `index_old_backup.html`，隨時可以換回：
```bash
cd /home/administrator/.openclaw/workspace/stock_web
cp index_old_backup.html index.html
```

### Q: 個人附註不填可以嗎？

A: 可以！完全選填。系統會自動填入推薦理由，附註只是讓你加入自己的想法。

### Q: CSV 可以在哪些軟體開啟？

A: 
- Excel（微軟）
- Google Sheets（雲端）
- Numbers（蘋果）
- LibreOffice Calc（免費）

### Q: 如果同一支股票分批買入，賣出時會賣哪一批？

A: 系統使用 **FIFO（先進先出）** 原則，會優先賣出最早買入的批次。

### Q: 資料會遺失嗎？

A: 不會！所有交易記錄都永久保存在 `stock_trades.db` 資料庫中。
