# 🦈 模擬交易系統使用指南

## 🎯 系統概述

這是一個**完整的模擬交易與績效追蹤系統**，讓你可以在不動用真實資金的情況下，練習股票買賣並追蹤長期績效。

---

## ✨ 主要功能

### 1. 📊 **投資組合頁面** - 即時管理持股

- **查看當前持股** - 所有模擬持倉一目了然
- **一鍵賣出** - 每一行都有「📉 賣出」按鈕
- **即時損益** - 自動計算每支股票的盈虧
- **快速操作** - 點擊按鈕即可模擬交易

### 2. 🏆 **V3.0 TOP 5** - 每日推薦股票

- **系統推薦** - AI 分析出的最佳買點
- **一鍵買入** - 每一行都有「📈 買入」按鈕
- **評分參考** - 技術分析 + 法人動向綜合評分
- **風險提示** - 低/中/高風險等級

### 3. 📊 **績效報告**（1年追蹤）

- **總損益** - 已實現 + 未實現
- **勝率統計** - 獲利交易比例
- **交易次數** - 買賣總次數
- **最佳/最差交易** - 找出表現最好與最差的股票

### 4. 💾 **記錄匯出**

- **CSV 格式** - 可在 Excel 開啟
- **完整記錄** - ID、日期、類型、代號、名稱、股數、價格、金額、備註
- **永久保存** - 資料庫自動保存，不會遺失

## 🚀 快速上手

### 場景一：從推薦清單買入

1. 打開首頁 http://localhost:8080
2. 點選「🏆 V3.0 TOP 5」分頁
3. 看到喜歡的股票，點擊「📈 買入」
4. 輸入股數（預設 1000 股）
5. 確認 → 完成！

**範例**：
```
系統推薦：聯邦銀 (2838)
評分：76
價格：$20.05
風險：低

→ 點「📈 買入」
→ 輸入 1000 股
→ 總金額：$20,050
```

### 場景二：賣出獲利

幾天後股票上漲，想要獲利了結：

1. 回到「💼 投資組合」分頁
2. 找到要賣的股票（例如：聯邦銀）
3. 點擊該行的「📉 賣出」按鈕
4. 輸入要賣出的股數
5. 確認 → 系統自動計算損益

**範例**：
```
聯邦銀 (2838)
買入價：$20.05
現價：$21.00
持有：1000 股

→ 點「📉 賣出」
→ 賣出 1000 股
→ 獲利：$950 (+4.74%) ✅
```

### 場景三：查看績效

1. 點選 header 的「📊 績效報告」按鈕
2. 彈出視窗顯示：
   - 總交易次數
   - 勝率
   - 已實現損益
   - 未實現損益
   - 總損益

### 場景四：匯出記錄

1. 點選 header 的「💾 匯出記錄」按鈕
2. 自動下載 CSV 檔案
3. 可用 Excel 開啟分析

## 📖 詳細操作說明

### 買入操作

#### 方法一：從推薦清單（最快）

```
🏆 V3.0 TOP 5 → 點「📈 買入」
```

- 系統自動帶入：代號、名稱、價格
- 你只需要輸入股數
- 一鍵完成

#### 方法二：手動買入（完整版）

```
💰 交易管理 → 買賣操作 → 填寫表單
```

- 適合買入「非推薦清單」的股票
- 可填寫更詳細的備註
- 可指定交易日期（回測用）

### 賣出操作

#### 從持倉清單賣出

```
💼 投資組合 → 找到股票 → 點「📉 賣出」
```

- 系統顯示當前持股數量
- 可選擇部分賣出或全部賣出
- 自動計算損益

## 📊 績效追蹤（1年）

系統自動記錄 **1年內** 的所有交易，包含：

### 自動統計

- ✅ **總交易次數** - 買入 + 賣出
- ✅ **勝率** - 獲利交易 / 總交易
- ✅ **已實現損益** - 已賣出的真實獲利/虧損
- ✅ **未實現損益** - 還沒賣的帳面損益
- ✅ **總損益** - 已實現 + 未實現

### 資料保存

- 📁 SQLite 資料庫：`stock_trades.db`
- 🗓️ 保存期限：永久（可手動清理）
- 💾 定期備份：建議每月備份一次
- 📤 隨時匯出：CSV 格式

### 1. 批次建倉

```javascript
// 瀏覽器 Console
// 批次買入 TOP 5
['2838', '2812', '2353', '5876', '2618'].forEach(symbol => {
    // 這裡可以寫腳本自動買入
```

### 2. 回測歷史

在「💰 交易管理」手動輸入過去的日期，可以模擬回測：

```
日期：2026-01-01
買入：台積電 @ $500
日期：2026-03-01
賣出：台積電 @ $520
→ 驗證策略是否有效
```

### 3. 組合策略測試

**策略 A**：只買 TOP 1  
**策略 B**：平均買入 TOP 5  
**策略 C**：只買低風險股票

分別建立不同的模擬帳戶（用備註區分），比較哪個策略表現更好。

## 💡 常見問題

### Q: 模擬買入後，價格會自動更新嗎？

A: 不會。目前需要手動點「📉 賣出」時輸入當前價格。未來版本會整合即時股價 API。

### Q: 可以刪除錯誤的交易嗎？

A: 可以。到「💰 交易管理 → 📜 交易歷史」找到該筆記錄，點「刪除」即可。

### Q: 績效報告只能看 1 年嗎？

A: 目前預設 1 年。可以修改 API 參數 `period` 來調整（例如：`?period=6months`）。

### Q: 匯出的 CSV 可以做什麼？

A: 可以在 Excel 做進階分析，例如：
- 繪製損益曲線圖
- 計算夏普比率
- 月度報酬統計
- 最大回撤分析

### Q: 可以同時模擬多個策略嗎？

A: 可以用「備註」欄位區分不同策略，例如：
- 備註：策略A-激進
- 備註：策略B-保守
