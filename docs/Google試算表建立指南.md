## 📋 步驟一：上傳 CSV 檔案到 Google Drive

### 方法 A：電腦上傳

1. **開啟 Google Drive**
   - 網址：https://drive.google.com
   - 登入你的 Google 帳號

2. **建立資料夾**
   ```
   新增資料夾 → 命名為「投資記錄」
   ```

3. **上傳 CSV 檔案**
   ```
   點選「新增」→「檔案上傳」
   選擇以下 4 個檔案：
   - investment_tracker_template.csv
   - 配息記錄表.csv
   - 持股總覽表.csv
   - 定期定額計畫表.csv
   ```

4. **轉換為 Google 試算表**
   ```
   右鍵點擊 CSV 檔案
   → 開啟方式
   → Google 試算表
   → 檔案 → 另存為 Google 試算表
   ```

---

### 方法 B：手機上傳

1. **下載 Google Drive App**
   - Android：Google Play 商店
   - iOS：App Store

2. **上傳檔案**
   ```
   開啟 App → 點選「+」
   → 上傳 → 選擇檔案
   → 選擇 4 個 CSV 檔案
   ```

3. **轉換為試算表**
   ```
   點選檔案 → 「⋮」更多
   → 開啟方式
   → Google 試算表
   ```

## 📊 步驟二：設定試算表格式

### 1. 交易明細表（investment_tracker）

#### 欄位格式設定

| 欄位名稱 | 格式類型 | 範例 |
|---------|---------|------|
| 交易日期 | 日期 | 2026-04-01 |
| 交易金額 | 貨幣 | $8,100 |
| 買入價格 | 數字(2位小數) | 81.00 |
| 手續費 | 貨幣 | $23 |
| 報酬率 | 百分比(2位小數) | -0.28% |

#### 設定步驟

```
1. 選取「交易金額」欄 → 格式 → 數字 → 貨幣（TWD）
2. 選取「報酬率」欄 → 格式 → 數字 → 百分比
3. 選取「買入價格」欄 → 格式 → 數字 → 自訂數字格式 → 0.00
```

#### 自動計算公式

**累積股數（假設在 L 欄）**
```
=SUMIF($C$2:$C, C2, $H$2:$H)
```
說明：加總同一股票的所有股數

**平均成本（假設在 M 欄）**
```
=SUMIF($C$2:$C, C2, $K$2:$K) / L2
```
說明：總成本 ÷ 累積股數

**未實現損益（假設在 P 欄）**
```
=O2 - (M2 * L2)
```
說明：目前市值 - 總成本

**報酬率（假設在 Q 欄）**
```
=P2 / (M2 * L2)
```
說明：未實現損益 ÷ 總成本

---

### 2. 配息記錄表

#### 欄位格式設定

| 欄位名稱 | 格式類型 | 範例 |
|---------|---------|------|
| 除息日 | 日期 | 2026-08-15 |
| 每股配息 | 貨幣 | $0.60 |
| 配息總額 | 貨幣 | $300 |
| 累計配息 | 貨幣 | $300 |

#### 自動計算公式

**配息總額（假設在 F 欄）**
```
=D2 * E2
```
說明：持有股數 × 每股配息

**實收金額（假設在 H 欄）**
```
=F2 - G2
```
說明：配息總額 - 扣繳稅額

**累計配息（假設在 I 欄）**
```
=SUM($H$2:H2)
```
說明：從第一筆累計到目前

---

### 3. 持股總覽表

#### 欄位格式設定

| 欄位名稱 | 格式類型 | 範例 |
|---------|---------|------|
| 平均成本 | 貨幣 | $81.16 |
| 總成本 | 貨幣 | $24,348 |
| 報酬率 | 百分比 | -0.20% |
| 佔比 | 百分比 | 30.4% |

#### 自動計算公式

**總成本（假設在 E 欄）**
```
=C2 * D2
```
說明：持有股數 × 平均成本

**目前市值（假設在 G 欄）**
```
=C2 * F2
```
說明：持有股數 × 目前市價

**未實現損益（假設在 H 欄）**
```
=G2 - E2
```
說明：目前市值 - 總成本

**報酬率（假設在 I 欄）**
```
=H2 / E2
```
說明：未實現損益 ÷ 總成本

**佔比（假設在 J 欄）**
```
=G2 / $G$總計行
```
說明：個股市值 ÷ 總市值

---

### 4. 定期定額計畫表

#### 欄位格式設定

| 欄位名稱 | 格式類型 | 範例 |
|---------|---------|------|
| 年月 | 文字 | 2026-04 |
| 所有金額欄 | 貨幣 | $5,000 |
| 累計總額 | 貨幣 | $85,000 |

#### 自動計算公式

**當月累積（假設在 I 欄）**
```
=SUM(C2:H2)
```
說明：加總當月所有投入

**累計總額（假設在 J 欄）**
```
=SUM($I$2:I2)
```
說明：從第一個月累計到目前

## 🎨 步驟三：美化試算表

### 1. 標題列格式

```
選取第一列（標題列）
→ 格式 → 文字 → 粗體
→ 格式 → 填滿顏色 → 淺藍色
→ 格式 → 對齊 → 置中對齊
→ 檢視 → 凍結 → 凍結第 1 列
```

### 2. 條件式格式設定

#### 損益顏色標示

```
選取「未實現損益」欄
→ 格式 → 條件式格式
→ 設定規則：
  - 如果大於 0 → 背景色：淺綠色
  - 如果小於 0 → 背景色：淺紅色
```

#### 報酬率顏色標示

```
選取「報酬率」欄
→ 格式 → 條件式格式
→ 設定規則：
  - 如果大於 5% → 背景色：深綠色
  - 如果大於 0% → 背景色：淺綠色
  - 如果小於 0% → 背景色：淺紅色
```

## 🔄 步驟五：自動化設定（進階）

### 1. 股價自動更新（Google Finance）

**公式範例：**
```
=GOOGLEFINANCE("TPE:2330", "price")
```

**在「目前市價」欄使用：**
```
=GOOGLEFINANCE("TPE:" & A2, "price")
```

**說明：**
- TPE = 台灣證券交易所代碼
- A2 = 股票代號欄位
- 自動抓取即時股價

**注意：**
- ⚠️ 可能有 15-20 分鐘延遲
- ⚠️ 部分ETF可能無法抓取

---

## 📞 疑難排解

### Q1：公式顯示錯誤？
A：檢查：
- 欄位參照是否正確（A2、B2等）
- 是否有空白儲存格
- 格式是否為「數字」

### Q2：手機無法編輯？
A：檢查：
- 是否有編輯權限
- 是否登入正確帳號
- App 是否為最新版本

### Q3：股價無法自動更新？
A：
- GOOGLEFINANCE 可能有延遲
- 部分ETF無法抓取
- 建議手動更新
