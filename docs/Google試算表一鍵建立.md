### 步驟 2：快速設定（複製貼上即可）

#### 工作表 1：交易明細表

**A. 設定標題列格式**
```
1. 選取第 1 列
2. 格式 → 文字 → 粗體
3. 格式 → 填滿顏色 → 淺藍色（#4A86E8）
4. 格式 → 對齊 → 水平對齊：置中
5. 檢視 → 凍結 → 凍結第 1 列
```

**B. 設定欄位格式**

| 欄位 | 格式設定 |
|------|---------|
| A 欄（交易日期） | 格式 → 數字 → 日期 |
| D-F 欄（價格） | 格式 → 數字 → 自訂 → 0.00 |
| H-K 欄（金額） | 格式 → 數字 → 貨幣 → TWD |
| Q 欄（報酬率） | 格式 → 數字 → 百分比 → 0.00% |

**C. 輸入公式**

在 L2 儲存格（累積股數）：
```
=SUMIF($C$2:$C, C2, $H$2:$H)
```

在 M2 儲存格（平均成本）：
```
=IF(L2=0, 0, SUMIF($C$2:$C, C2, $K$2:$K) / L2)
```

在 O2 儲存格（目前市值）：
```
=L2 * N2
```

在 P2 儲存格（未實現損益）：
```
=O2 - (L2 * M2)
```

在 Q2 儲存格（報酬率）：
```
=IF(L2=0, 0, P2 / (L2 * M2))
```

**D. 複製公式到下方**
```
選取 L2:Q2
→ 拖曳右下角藍點到第 100 列
→ 公式自動複製
```

#### 工作表 2：配息記錄表

**A. 新增工作表**
```
點選左下角「+」
→ 命名：配息記錄
```

**B. 輸入標題列**
```

D1: 股票名稱
E1: 持有股數
F1: 每股配息
G1: 配息總額
H1: 扣繳稅額
I1: 實收金額
J1: 累計配息
K1: 備註
```

**C. 輸入公式**

在 G2 儲存格（配息總額）：
```
=E2 * F2
```

在 I2 儲存格（實收金額）：
```
=G2 - H2
```

在 J2 儲存格（累計配息）：
```
=SUM($I$2:I2)
```

#### 工作表 3：持股總覽表

**A. 新增工作表**
```
點選左下角「+」
→ 命名：持股總覽
```

**B. 輸入標題列**
```
A1: 股票代號
B1: 股票名稱
C1: 持有股數
D1: 平均成本
E1: 總成本
F1: 目前市價
G1: 目前市值
H1: 未實現損益
I1: 報酬率(%)
J1: 佔比(%)
K1: 目標比例(%)
L1: 調整建議
```

**C. 輸入公式**

在 E2 儲存格（總成本）：
```
=C2 * D2
```

在 G2 儲存格（目前市值）：
```
=C2 * F2
```

在 H2 儲存格（未實現損益）：
```
=G2 - E2
```

在 I2 儲存格（報酬率）：
```
=IF(E2=0, 0, H2 / E2)
```

在 J2 儲存格（佔比）：
```
=IF($G$總計=0, 0, G2 / $G$總計)
```
（記得將「總計」改為實際總計列的行號）

## 🔧 常用公式速查表

### 基本計算

```
加總：=SUM(A1:A10)
平均：=AVERAGE(A1:A10)
計數：=COUNT(A1:A10)
最大值：=MAX(A1:A10)
最小值：=MIN(A1:A10)
```

### 條件計算

```
條件加總：=SUMIF(範圍, 條件, 加總範圍)
範例：=SUMIF(C:C, "0050", H:H)

條件計數：=COUNTIF(範圍, 條件)
範例：=COUNTIF(C:C, "0050")

條件平均：=AVERAGEIF(範圍, 條件, 平均範圍)
```

### 邏輯判斷

```
IF：=IF(條件, 真值, 假值)
範例：=IF(A1>0, "獲利", "虧損")

IFS（多條件）：=IFS(條件1, 值1, 條件2, 值2, ...)
範例：=IFS(A1>10, "大漲", A1>0, "上漲", A1<0, "下跌")
```

### 查詢功能

```
VLOOKUP：=VLOOKUP(查詢值, 範圍, 欄位, FALSE)
範例：=VLOOKUP("0050", A:D, 2, FALSE)

INDEX + MATCH：=INDEX(回傳範圍, MATCH(查詢值, 查詢範圍, 0))
```

## ⚠️ 常見問題

### Q1：公式顯示 #REF! 錯誤？
A：檢查儲存格參照是否正確（如 A2、B2）

### Q2：公式顯示 #DIV/0! 錯誤？
A：除數為 0，加上 IF 判斷：`=IF(B2=0, 0, A2/B2)`

### Q3：日期顯示為數字？
A：格式 → 數字 → 日期

### Q4：手機無法編輯？
A：檢查是否有編輯權限、是否登入正確帳號
