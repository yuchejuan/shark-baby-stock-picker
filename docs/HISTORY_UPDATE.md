# 📜 歷次買賣頁面更新說明

## 🎯 更新內容

在「📜 歷次買賣」頁面新增三個欄位：

1. **買入日期** - 顯示這批股票最早的買入日期（FIFO 配對）
2. **持有天數** - 從買入到賣出的天數
3. **損益** - 這筆交易的實際獲利/虧損

## 🧮 損益計算邏輯

### FIFO（先進先出）配對

系統會自動配對買賣記錄，使用 **FIFO 原則**：

#### 範例 1：單批買賣
```
買入：2026/3/25  1000股 @ $27.15
賣出：2026/3/30  1000股 @ $28.50

損益計算：
賣出收入: 1000 × $28.50 = $28,500
買入成本: 1000 × $27.15 = $27,150
損益: $28,500 - $27,150 = +$1,350 ✅

買入日期: 2026/3/25
持有天數: 5 天
```

#### 範例 2：分批買入，一次賣出
```
買入 1：2026/3/25  1000股 @ $27.15
買入 2：2026/3/27   500股 @ $28.00
賣出：  2026/3/30  1000股 @ $29.00

損益計算（FIFO）：
賣出收入: 1000 × $29.00 = $29,000
買入成本: 1000 × $27.15 = $27,150 （全部從第 1 批）
損益: $29,000 - $27,150 = +$1,850 ✅

買入日期: 2026/3/25 （最早的買入）
持有天數: 5 天
```

#### 範例 3：分批買入，部分賣出
```
買入 1：2026/3/25  1000股 @ $27.15
買入 2：2026/3/27  1000股 @ $28.00
賣出：  2026/3/30  1500股 @ $29.00

損益計算（FIFO）：
賣出收入: 1500 × $29.00 = $43,500
買入成本: 
  - 1000股 @ $27.15 = $27,150 （第 1 批全部）
  -  500股 @ $28.00 = $14,000 （第 2 批部分）
  - 總成本 = $41,150
損益: $43,500 - $41,150 = +$2,350 ✅

買入日期: 2026/3/25 （最早的買入）
持有天數: 5 天
```

匯出的 CSV 檔案也包含新欄位：

```csv

2026/3/25,買入,2353,宏碁,1000,27.15,27150,,,,"MACD黃金交叉 + KD超賣"
2026/3/30,賣出,2353,宏碁,1000,28.50,28500,2026/3/25,5,1350,"模擬賣出"
```

### 在 Excel 分析

可以使用 Excel 的公式和圖表：

**計算總損益**：
```excel
=SUMIF(B:B, "賣出", J:J)
```

**計算平均持有天數**：
```excel
=AVERAGEIF(B:B, "賣出", I:I)
```

**勝率統計**：
```excel
=COUNTIF(J:J, ">0") / COUNTIF(B:B, "賣出")
```

## 🔧 技術實現

### 前端邏輯（JavaScript）

```javascript
async function loadHistory() {
    // 1. 載入所有交易
    const trades = await fetch('/api/trades').json();
    
    // 2. 建立買入記錄 Map
    const buyMap = {};
    trades.forEach(t => {
        if (t.type === 'buy') {
            buyMap[t.symbol].push({
                date: t.date,
                price: t.price,
                remainingShares: t.shares
            });
        }
    });
    
    // 3. 計算每筆賣出的損益
    trades.forEach(t => {
        if (t.type === 'sell') {
            // FIFO 配對
            const buys = buyMap[t.symbol];
            let sellShares = t.shares;
            let totalCost = 0;
            let firstBuyDate = null;
            
            for (let buy of buys) {
                if (sellShares <= 0) break;
                if (buy.remainingShares <= 0) continue;
                
                if (!firstBuyDate) firstBuyDate = buy.date;
                
                const sharesToSell = Math.min(sellShares, buy.remainingShares);
                totalCost += sharesToSell * buy.price;
                buy.remainingShares -= sharesToSell;
                sellShares -= sharesToSell;
            }
            
            // 計算損益
            const sellRevenue = t.shares * t.price;
            const profit = sellRevenue - totalCost;
            
            // 計算持有天數
            const daysHeld = (new Date(t.date) - new Date(firstBuyDate)) / (1000*60*60*24);
        }
    });
}
```

## 📌 重點提示

1. **買入記錄** - 不顯示買入日期、持有天數、損益（這三欄顯示 `-`）
2. **賣出記錄** - 自動配對最早的買入，計算損益
3. **FIFO 原則** - 優先賣出最早買入的批次
4. **CSV 匯出** - 包含所有欄位，可在 Excel 分析
