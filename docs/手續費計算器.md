## 📊 手續費計算公式

### 買入成本

```
買入成本 = 股價 × 股數 × (1 + 手續費率)
手續費率 = 0.1425%
最低手續費 = 20元
```

### 賣出成本

```
賣出所得 = 股價 × 股數 × (1 - 手續費率 - 交易稅率)
手續費率 = 0.1425%
交易稅率 = 0.3%（個股）/ 0.1%（ETF）
最低手續費 = 20元
```

## 🧮 手續費計算機（快速版）

### 買入成本計算

```
投入金額：_________ 元
手續費率：0.1425%
最低手續費：20元

計算：
1. _________ × 0.1425% = _______ 元
2. 若 < 20元，則手續費 = 20元
3. 總成本 = _________ + _______ = _______ 元
```

### 範例

```
投入金額：10,000 元
手續費：10,000 × 0.1425% = 14.25元
實際手續費：20元（最低限制）
總成本：10,000 + 20 = 10,020元
手續費比例：20 / 10,000 = 0.2%
```
