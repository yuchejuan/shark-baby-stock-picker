# 🦈 Google Apps Script 自動建立教學

**適用對象**：阿哲  
**預計時間**：3-5 分鐘

---

## 📋 完整步驟（跟著做就對了）

### 步驟 1：開啟 Google 試算表

1. **開啟瀏覽器**（Chrome、Safari、Edge 都可以）

2. **前往 Google 試算表**
   ```
   網址：https://sheets.google.com
   ```

3. **登入 Google 帳號**（如果還沒登入）

4. **建立新試算表**
   ```
   點選左上角「空白」（大大的 + 號）
   ```

---

### 步驟 2：開啟 Apps Script 編輯器

1. **點選「擴充功能」選單**
   ```
   上方選單列：擴充功能
   ```

2. **點選「Apps Script」**
   ```
   擴充功能 → Apps Script
   ```

3. **會開啟新分頁**（Apps Script 編輯器）
   ```
   預設會有一些程式碼：
   function myFunction() {
     ...
   }
   ```

---

### 步驟 3：貼上腳本

1. **全選刪除預設程式碼**
   ```
   按 Ctrl + A（全選）
   按 Delete（刪除）
   ```

2. **複製我的腳本**
   ```
   開啟檔案：試算表自動建立腳本.js
   全選複製（Ctrl + A → Ctrl + C）
   ```

3. **貼上到 Apps Script**
   ```
   在 Apps Script 編輯器中
   按 Ctrl + V（貼上）
   ```

4. **儲存腳本**
   ```
   點選「儲存專案」圖示（💾）
   或按 Ctrl + S
   ```

5. **命名專案**（可選）
   ```
   點選左上角「未命名的專案」
   改名為：阿哲投資記錄
   ```

---

### 步驟 4：執行腳本

1. **選擇要執行的函式**
   ```
   在上方下拉選單中
   選擇：createInvestmentTracker
   （預設可能是 myFunction，要改過來）
   ```

2. **點選「執行」按鈕**
   ```
   點選上方的「▶ 執行」按鈕
   ```

3. **第一次會要求授權**
   ```
   會跳出「需要授權」視窗
   點選「審查權限」
   ```

4. **選擇 Google 帳號**
   ```
   選擇你的帳號
   ```

5. **確認權限**
   ```
   會看到「Google 尚未驗證這個應用程式」
   點選左下角「進階」
   點選「前往『阿哲投資記錄』（不安全）」
   （這是你自己的腳本，安全無虞）
   ```

6. **允許權限**
   ```
   會列出需要的權限：
   ✓ 查看及管理你的試算表
   
   點選「允許」
   ```

---

### 步驟 5：等待完成

1. **腳本開始執行**
   ```
   下方會顯示「執行中...」
   預計 10-15 秒
   ```

2. **會跳出進度訊息**
   ```
   🦈 開始建立投資記錄表格...
   
   預計 10-15 秒完成，請稍候！
   ```
   點選「確定」

3. **完成後會跳出訊息**
   ```
   🎉 完成！
   
   已建立以下工作表：
   ✅ 交易明細表
   ✅ 配息記錄表
   ✅ 持股總覽表
   ✅ 定期定額計畫表
   ✅ 總覽儀表板
   
   現在可以開始記錄交易了！🦈
   ```
   點選「確定」

---

### 步驟 6：查看結果

1. **關閉 Apps Script 分頁**
   ```
   直接關閉 Apps Script 分頁
   ```

2. **回到試算表分頁**
   ```
   會看到已經建立好 5 個工作表：
   - 總覽儀表板（預設開啟）
   - 交易明細表
   - 配息記錄表
   - 持股總覽表
   - 定期定額計畫表
   ```

3. **重新命名試算表**
   ```
   點選左上角「未命名的試算表」
   改名為：阿哲投資記錄 - 2026
   自動儲存
   ```

---

## ⚠️ 常見問題

### Q1：執行時出現錯誤？

**A：檢查以下項目：**
1. 是否選擇了 `createInvestmentTracker` 函式？
2. 是否完整貼上所有程式碼？
3. 是否授權完成？

**解決方法：**
- 重新執行一次
- 或重新貼上腳本

---

### Q2：沒有出現「執行」按鈕？

**A：可能是：**
1. 沒有儲存腳本（按 Ctrl + S）
2. 沒有選擇函式（選擇 createInvestmentTracker）

---

### Q3：授權時卡住？

**A：**
1. 確認已登入 Google 帳號
2. 點選「進階」→「前往『阿哲投資記錄』」
3. 點選「允許」

---

### Q4：執行完成但沒有工作表？

**A：**
1. 檢查是否有錯誤訊息
2. 重新整理試算表頁面（F5）
3. 或重新執行一次腳本

---

### Q5：公式沒有計算？

**A：**
1. 檢查公式是否正確貼入
2. 檢查資料格式（數字是數字、日期是日期）
3. 手動重新輸入公式
