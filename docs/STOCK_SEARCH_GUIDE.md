# 🔍 股票查詢功能使用指南

## 🎯 功能概述

在「📊 TOP 選股」頁面新增**股票查詢功能**，讓你可以：
- 輸入任意股票代號
- 查看即時價格
- 查看完整的六大技術指標分析
- 獲得操作建議

---

## 📊 查詢結果包含

### 1. 基本資訊
- 股票代號與名稱
- 現價
- 綜合評分（0-100 分）
- 訊號（買點/賣點/中性）
- 推薦理由

### 2. 六大技術指標詳細分析

#### 1️⃣ RSI（相對強弱）
- 顯示數值（0-100）
- 判斷：超賣 / 偏低 / 中性 / 偏高 / 超買
- 解讀說明

#### 2️⃣ MACD（趨勢方向）
- 顯示趨勢（多頭 / 空頭 / 中性）
- 趨勢解讀

#### 3️⃣ KD 指標（短線買賣）
- 顯示狀態（超賣 / 中性 / 超買）
- 短線機會判斷

#### 4️⃣ 移動平均線（MA）
- MA5（週線）
- MA20（月線）
- MA60（季線）
- 趨勢判斷（多頭排列 / 空頭排列）

#### 5️⃣ 價格位置
- 現價 vs MA20
- 現價 vs MA60
- 價格強弱判斷

#### 6️⃣ 綜合訊號
- 買點 / 賣點 / 中性
- 訊號解讀

### 3. 操作建議
- 根據評分給出建議
- 多指標共振提示
- 停損建議
- 風險提示

### 4. 快速買入
- 一鍵模擬買入按鈕
- 自動帶入推薦理由

---

## 🎮 使用方法

### 步驟 1：打開 TOP 選股頁面
```
http://localhost:8080
→ 點「📊 TOP 選股」分頁
```

### 步驟 2：輸入股票代號
```
在「🔍 股票查詢」框輸入代號
例如：2330、2353、2454
```

### 步驟 3：點擊查詢
```
點「🔍 查詢」按鈕
```

### 步驟 4：查看結果
```
系統會顯示：
- 基本資訊
- 六大技術指標
- 操作建議
- 買入按鈕
```

## 💡 技術指標解讀速查表

### RSI（相對強弱）
| 數值 | 判斷 | 說明 |
|------|------|------|
| < 30 | 超賣 🟢 | 嚴重低估，可能反彈 |

| 40-60 | 中性 ⚪ | 正常波動區間 |
| 60-70 | 偏高 🟠 | 注意回檔風險 |
| > 70 | 超買 🔴 | 可能回檔 |

|------|------|

## ⚠️ 注意事項

### 1. 資料範圍限制
- 目前只能查詢**今日篩選範圍內**的股票
- 範圍：20-50 元區間
- 如果查不到，可能：
  - 股價不在 20-50 元
  - 技術面評分太低（< 50 分）
  - 股票代號輸入錯誤

### 2. 資料更新頻率
- 每天早上 06:00 更新
- 盤中不會即時更新指標
- 但價格會每 20 分鐘更新（如果持有）

### 3. 技術分析不是萬能
- 只能參考，不能保證獲利
- 建議搭配基本面分析
- 務必設定停損

### 4. 模擬交易提醒
- 這是模擬交易，不是真實投資
- 用來學習和測試策略
- 真實投資前要謹慎評估

## 🚀 進階技巧

### 技巧 1：多指標共振篩選

```
查詢股票時，重點看：
1. MACD = 多頭 ✅
2. KD = 超賣 ✅
3. MA = 多頭排列 ✅
4. RSI < 60 ✅

→ 四項都符合 = 絕佳買點！
```

### 技巧 2：賣出時機判斷

```
持股後定期查詢：
1. RSI > 70（超買）
2. KD = 超買
3. MACD 轉空頭
4. 跌破 MA20

→ 出現 2 項以上 = 考慮賣出
```

### 技巧 3：建立觀察清單

```
每天查詢 5-10 支感興趣的股票：
1. 記錄評分和指標
2. 觀察變化趨勢
3. 等待最佳買點
4. 多指標共振時買入
```

## 🔧 故障排除

### 問題 1：查詢後沒有反應

**可能原因**：
- 網路問題
- daily_report.json 未載入

**解決方法**：
1. 檢查網路連線
2. 重新整理網頁（Ctrl+Shift+R）
3. 確認檔案存在：`ls stock_web/daily_report.json`

---

### 問題 2：找不到股票

**可能原因**：
- 股票代號錯誤
- 股票不在篩選範圍

**解決方法**：
1. 確認代號正確（例如：2330，不是 02330）
2. 檢查股價是否在 20-50 元區間
3. 試試其他股票

---

### 問題 3：顯示錯誤訊息

**解決方法**：
1. 查看瀏覽器 Console（F12）
2. 檢查 daily_report.json 格式
3. 重新產生選股報告：`go run daily_stock_picker_v2.go`
